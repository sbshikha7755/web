HTML

HTML(Hyper Text Markup Language) is standrad language use to creat web pages.

It structures web content using elements such as headings, paragraphs, images, links, and more.

KEY FEATURES OF HTML:-

  • Simple & Easy to Learn:- HTML uses simple tags to define content.
  • Platform Independent:- Works on all browsers and operating systems.
  • Hyperlinking:- Connects diffrent web pages using tags.
  • Multimedia Support:- Embeds images, videos, and audio.

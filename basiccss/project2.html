How does this work?

the key to a lot of this is leveraging using tablets with a role="presentation"attribution to make decorated section without css . If you view source on the page,you'll see that there are a LOT of tables. The table tag typicly is used to disply tabular data and should NOT be used as a layout method. However , since I am not allowed to use CSS. I am pretty much relying on tables for this purpose. The presentation role tells the browser to not expose the table and its tabular childern in the accessibilty tree. This will make it easre to navigate for assistive technology.
